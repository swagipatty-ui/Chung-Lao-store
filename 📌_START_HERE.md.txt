## ⚡ Super Quick Start (30 minutes)

### 1. Get Your Credentials (10 min)

**From Telegram:**
- Search `@BotFather`
- Send `/newbot`
- Copy your **API Token**

**From ceoacc.com:**
- Login to your account
- Get your **API Username**
- Get your **API Password**
- Get your **USDT Wallet Address**

**Your Telegram ID:**
- Search `@userinfobot`
- Copy your **User ID**

### 2. Edit the Bot (5 min)

Open `telegram_bot_LIVE.py` and change these 6 values (lines 20-36):

```python
TELEGRAM_TOKEN = 'YOUR_BOT_TOKEN_HERE'      # From @BotFather
ADMIN_ID = 123456789                         # Your Telegram ID

API_USERNAME = 'your_api_username'           # From ceoacc.com
API_PASSWORD = 'your_api_password'           # From ceoacc.com

USDT_ADDRESS = 'YOUR_WALLET_ADDRESS'         # For deposits
ADMIN_USERNAME = '@YourHandle'                # Your Telegram handle
```

**That's it!** Don't change anything else.

### 3. Install & Run (5 min)

```bash
pip install python-telegram-bot requests
python telegram_bot_LIVE.py
```

### 4. Test (5 min)

In Telegram, find your bot and send:
```
/start
/balance
/products
```

Everything working? ✅ **YOU'RE LIVE!**

### 5. Keep Running (5 min)

**Option A - Simple:**
Keep terminal open

**Option B - Background (Linux/Mac):**
```bash
nohup python telegram_bot_LIVE.py > bot.log 2>&1 &
```

**Option C - Best (Linux/VPS):**
```bash
screen -S bot
python telegram_bot_LIVE.py
# Press Ctrl+A then D
```
