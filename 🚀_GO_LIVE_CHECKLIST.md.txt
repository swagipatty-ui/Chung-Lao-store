#### **Option B: Background (Linux/Mac)**
Run in the background:
```bash
nohup python telegram_bot_LIVE.py > bot.log 2>&1 &
```

Check if running:
```bash
ps aux | grep telegram_bot_LIVE
```

Stop if needed:
```bash
pkill -f telegram_bot_LIVE
```

---

#### **Option C: Screen/Tmux (Linux/VPS - BEST)**
```bash
# Start in screen
screen -S bot
python telegram_bot_LIVE.py

# Detach: Press Ctrl+A then D
# Resume: screen -r bot
```

---

#### **Option D: systemd Service (Linux - ENTERPRISE)**

Create `/etc/systemd/system/telegram-bot.service`:
```ini
[Unit]
Description=Chung Lao Store Bot
After=network.target

[Service]
Type=simple
User=YOUR_USERNAME
WorkingDirectory=/path/to/bot
ExecStart=/usr/bin/python3 /path/to/telegram_bot_LIVE.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Enable:
```bash
sudo systemctl enable telegram-bot
sudo systemctl start telegram-bot
sudo systemctl status telegram-bot
```

## 🔧 Troubleshooting

### "ModuleNotFoundError: No module named 'telegram'"
```bash
pip install python-telegram-bot requests
```

### "TELEGRAM_TOKEN not set"
- Open `telegram_bot_LIVE.py`
- Find `TELEGRAM_TOKEN = 'YOUR_BOT_TOKEN_HERE'`
- Replace with actual token from @BotFather
- Save file

### "Products not loading"
- Check API username/password in config
- Verify ceoacc.com credentials work
- Check internet connection
- Check `bot.log` for errors:
  ```bash
  tail -50 bot.log | grep -i error
  ```

### "Balance not updating"
- Check `users.json` exists
- Check file permissions:
  ```bash
  chmod 666 users.json
  ```
- Restart bot if corrupted

### "Bot not responding"
- Is bot process running?
  ```bash
  ps aux | grep telegram_bot_LIVE
  ```
- Check logs:
  ```bash
  tail -50 bot.log
  ```
- Restart bot:
  ```bash
  pkill -f telegram_bot_LIVE
  python telegram_bot_LIVE.py
  ```
